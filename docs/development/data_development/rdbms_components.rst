RDBMS Components
================

RDBMS Drop (Legacy)
-------------------

:class:`RDBMSDrop <dlg.drop.RDBMSDrop>` is a generic relational database storage drop that uses SQL
to insert and select data. Instead of using the getIO() interface the methods `insert` and `select`
are available.
