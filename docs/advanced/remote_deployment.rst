Remote Subgraph Deployment
==========================

.. warning:: 
    This section discusses alpha features and procedures that are still being actively 
    tested by the developers. 

Subgraphs may be used to deploy running long-term remote graphs on a local machine. 

This page outlines: 

- Add existing graph to the subgraph 
- Running palettegen to create the SlurmClient palette
- Reminder to set things up using the configuration files. 
- Configure the rest of the client 
- Run test deployment
