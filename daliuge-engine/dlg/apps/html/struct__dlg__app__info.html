Detailed Description

The representation of an application instance. Apart from its UID and OID, an application has an array of inputs (with n_inputs elements), streaming_inputs (with n_streaming_inputs) elements, and outputs (with n_outputs elements). If the application works in a streaming mode, then it is also given the running and done callbacks to signal the framwork that it started working, and that it is done with its work, respectively. Finally, a data pointer holds application-specific data.
